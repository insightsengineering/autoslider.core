# Preprocess t_ds function

Preprocess t_ds function

## Usage

``` r
preprocess_t_ds(
  df,
  levels = c("Alive: On Treatment", "Alive: In Follow-up", "<Missing>"),
  labels = c("Alive: On Treatment", "Alive: In Follow-up", "<Missing>")
)
```

## Arguments

- df:

  Input dataframe

- levels:

  factor levels

- labels:

  factor labels

## Value

dataframe
