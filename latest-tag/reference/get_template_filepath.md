# Retrieve Template File Paths

Retrieve Template File Paths

## Usage

``` r
get_template_filepath(package = "autoslider.core", full.names = FALSE)
```

## Arguments

- package:

  A character string specifying the name of the package to search.

- full.names:

  If \`TRUE\`, returns the full path to each file. If \`FALSE\`, returns
  only the file names.

## Value

A character vector of template file names or paths, depending on
\`full.names\`.
