# Add decorated flextable to slides

Add decorated flextable to slides

## Usage

``` r
table_to_slide(
  ppt,
  content,
  decor = TRUE,
  layout = "Title and Content",
  table_loc = ph_location_type("body"),
  usernotes = "",
  ...
)
```

## Arguments

- ppt:

  Slide

- content:

  Content to be added

- decor:

  Should table be decorated

- layout:

  layout from theme

- table_loc:

  Table location

- usernotes:

  User notes

- ...:

  additional arguments

## Value

Slide with added content
