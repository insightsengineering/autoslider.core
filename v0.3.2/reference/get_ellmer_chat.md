# Get an \`ellmer\` chat API with given platform

Get an \`ellmer\` chat API with given platform

## Usage

``` r
get_ellmer_chat(
  platform = "deepseek",
  base_url = "https://api.deepseek.com",
  api_key = get_deepseek_key(),
  model = "deepseek-chat"
)
```

## Arguments

- platform:

  Platform provider

- base_url:

  Base url

- api_key:

  API key

- model:

  Model of choice
