# Create location container to center the figure, based on ppt size and user specified figure size

Create location container to center the figure, based on ppt size and
user specified figure size

## Usage

``` r
center_figure_loc(fig_width, fig_height, ppt_width, ppt_height)
```

## Arguments

- fig_width:

  Figure width

- fig_height:

  Figure height

- ppt_width:

  Slide width

- ppt_height:

  Slide height

## Value

Location for a placeholder from scratch
