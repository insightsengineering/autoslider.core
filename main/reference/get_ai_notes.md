# Update footnote with AI response

Update footnote with AI response

## Usage

``` r
get_ai_notes(outputs, prompt_list, platform, base_url, api_key, model)
```

## Arguments

- outputs:

  Output objects

- prompt_list:

  List of prompt

- platform:

  Platform provider

- base_url:

  Base url

- api_key:

  API key

- model:

  Model of choice
