# Covert rtables object to flextable

Covert rtables object to flextable

## Usage

``` r
# S3 method for class 'VTableTree'
to_flextable(x, table_format = orange_format, ...)
```

## Arguments

- x:

  rtable(VTableTree) object

- table_format:

  a function that decorate a flextable and return a flextable
