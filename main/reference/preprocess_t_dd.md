# Preprocess t_dd function

Preprocess t_dd function

## Usage

``` r
preprocess_t_dd(
  df,
  levels = c("PROGRESSIVE DISEASE", "ADVERSE EVENT", "OTHER", "<Missing>"),
  labels = c("Progressive Disease", "Adverse Events", "Other", "<Missing>")
)
```

## Arguments

- df:

  Input dataframe

- levels:

  factor levels

- labels:

  factor labels

## Value

dataframe
