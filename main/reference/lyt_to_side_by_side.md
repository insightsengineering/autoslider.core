# Build side by side layout by cbind

Build side by side layout by cbind

## Usage

``` r
lyt_to_side_by_side(lyt, anl, side_by_side = NULL)
```

## Arguments

- lyt:

  layout object

- anl:

  analysis data object

- side_by_side:

  A logical value indicating whether to display the data side by side.

## Value

An \`rtables\` layout
